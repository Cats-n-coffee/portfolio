Frontend Mentor - Manage Landing Page

A landing page with an image slider to display user reviews. Like the rest of the page, this image slider is responsive and displays only one slide at a time on mobile with a colored bubble to indicate the slide number. Achieved with HTML, Javascript and SCSS. A challenge by Frontend Mentor.
